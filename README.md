## 📌 درباره پروژه (ABOUT)
نرم‌افزار **CHANNEL EDITE VIP3R** یک ابزار حرفه‌ای تحت سیستم‌عامل ویندوز است که برای ویرایش فایل‌های دیتابیس رسیورهای دارای پردازنده GX (با فرمت‌های `.sdx` و `.chl`) طراحی شده است. 

در حالت عادی، ویرایش فایل‌های دیتابیس رسیور به دلیل وجود قفل‌های امنیتی و Checksum بسیار دشوار است و باعث خراب شدن (CORRUPTION) فایل می‌شود. اما این نرم‌افزار با دور زدن هوشمندانه این محدودیت‌ها از طریق پردازش متن خام (RAW STRING ENGINE)، به شما اجازه می‌دهد بدون هیچ‌گونه نگرانی، شبکه‌ها، ماهواره‌ها و فرکانس‌های خود را مدیریت کنید.

## ✨ ویژگی‌های کلیدی (FEATURES)
* **مدیریت سه‌گانه و کامل:** کنترل ۱۰۰ درصدی روی لیست ماهواره‌ها (SATELLITES)، فرکانس‌ها (TRANSPONDERS) و کانال‌ها (CHANNELS).
* **استخراج هوشمند (SMART EXTRACTOR):** قابلیت فوق‌العاده کاربردی برای کپی کردن لیست فرکانس‌ها از سایت‌های مرجع (مثل LYNGSAT و FLYSAT) و تزریق خودکار آن‌ها به دیتابیس تنها با یک کلیک!
* **حذف ایمن (SOFT DELETE):** برای جلوگیری از کرش کردن رسیور و حفظ ساختار دیتابیس، آیتم‌های حذفی در پس‌زمینه صفر (0) می‌شوند اما بلافاصله از رابط کاربری برنامه ناپدید می‌گردند تا محیط کاری شما تمیز بماند.
* **رابط کاربری مدرن (MODERN UI):** محیطی چشم‌نواز طراحی شده با PYTHON و PYQT6 با پشتیبانی از قابلیت کشیدن و رها کردن (DRAG & DROP) و انتخاب گروهی ده‌ها آیتم (MULTI-SELECT).
* **سیستم کدگذاری رنگی (COLOR CODING):** تشخیص سریع و بصری وضعیت فرکانس‌ها و کانال‌ها (رنگ سبز/آبی ملایم برای آیتم‌های جدید و رنگ نارنجی برای آیتم‌های ویرایش‌شده).

## 🚀 نحوه استفاده (USAGE)
۱. در همین صفحه و از بخش **Releases**، آخرین نسخه فایل `CHANNEL_EDITE_VIP3R.exe` را دانلود کنید (بدون نیاز به نصب پیش‌نیاز).
۲. فایل دیتابیس رسیور خود (که با فلش ממوری بک‌آپ گرفته‌اید) را در برنامه باز کنید `OPEN FILE`.
۳. تغییرات دلخواه خود را روی کانال‌ها و فرکانس‌ها اعمال کرده و در نهایت روی `SAVE CHANGES` کلیک کنید.
۴. فایل خروجی را مجدداً به فلش مموری منتقل کرده و در رسیور خود به‌روزرسانی (Update/Load) کنید.
